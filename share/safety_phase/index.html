3. Method

We consider a set of concepts $\mathcal{C} = (z_1, z_2, \cdots)$ where each concept $c$ is related to a human-oriented concept, such as harmless or safety. Each concept $z$ has examples, a set of passages \(\mathcal{Y}_{c}\), where passage \(y = (y_1, y_2, \cdots) \in \mathcal{Y}_{c}\) consists of tokens $y_i$ and includes the semantic meaning of concept $c$. We use notation $c \perp câ€™$ when two concepts $c$ and $câ€™$ can not exist together, e.g., safety concept and violence concept.

We define two terms phase and *phase transition$ as follows.

Definition: Phase

A $c$-phase is a subset of time steps $[t_{begin}, t_{end}]$ where the generated content \(({y}_{t_{begin}}, \cdots, {y}_{t_{end}})\) with \({y}_{t_{k}} \sim P_{\theta}(\cdot \vert y_1, \cdots, y_{t_{k}-1})\) can be included in the set of concept-related passages \(\mathcal{Y}_{c}\).

The definition of phase assumes a passage $y$. The inclusion of an element in the set element in $\mathcal{Y}_{c}$ can be either determined by a human-annotator or text similarity scores with the examples in the set.

Definition: Phase Emergence

A phase emergence is the time step when the generation includes new concept $câ€™$ as a phase.
